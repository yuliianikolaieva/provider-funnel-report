# Партнерські огляди на Bolt Food — Україна

Багатосторінковий партнерський огляд по містах і закладах України. Дані тягнуться напряму з **Databricks** (`fact_provider_weekly`, `fact_provider_non_additive_weekly`, `fact_order_delivery`) і генеруються скриптом — звіт можна переоновлювати щомісяця.

**Живий звіт:** https://yuliianikolaieva.github.io/provider-funnel-report/

## Структура
- Вкладки партнерів: VARUS, Копійка, Локо, Hop Hey, Rukavichka, Beer Market.
- На кожній — перемикач **місяців** (січень–травень 2026, лише з даними).
- Окрема вкладка **«Порівняння»** — динаміка місяців по всіх партнерах (GMV, замовлення, середній чек, конверсія, нові клієнти, утримання) + зведена таблиця.
- Кожну сторінку можна окремо зберегти у PDF (блоки не розриваються).

## Метрики
- Оборот (GMV), доставлені замовлення, активні заклади, **середній чек** (у т.ч. на рівні міста та закладу).
- Воронка: перегляд меню → кошик → доставлено; **відвал на етапі кошика** (додав товар, але не замовив).
- Споживачі: активні клієнти, нові клієнти, **частота замовлень**, **повторні клієнти**, **утримання 30 днів**, повернення до бренду, розмір кошика, покази.

## Як оновити дані
1. `python3 build_data.py` — тягне свіжі дані з Databricks у `report_data.json`.
2. `python3 build_html.py` — збирає `index.html`.
3. Закомітити й запушити.
